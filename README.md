# Jadwal Dosen BINGPRO

📅 Aplikasi jadwal mengajar dosen BINGPRO (Bahasa Inggris Profesional) - Politeknik Pertanian Negeri Payakumbuh.

## Fitur
- 📋 Tampilan kartu & tabel
- 🔍 Pencarian dosen, mata kuliah, ruang
- 📚 Filter berdasarkan prodi & dosen
- 📊 Statistik (jumlah dosen, MK, SKS)
- 🌙 Mode gelap/terang
- 📱 Installable PWA (bisa ditambahkan ke layar utama HP)
- 🔄 Data live dari Google Sheets

## Cara Install di HP
1. Buka link di Chrome/Safari
2. Tap menu (3 titik) → "Install app" / "Add to Home Screen"
3. Aplikasi akan muncul di layar utama
